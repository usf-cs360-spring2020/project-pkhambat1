Summary

This website takes a look into a large dataset about US home prices over time. The dataset is provided by Zillow - a popular online real estate company.

üè† We're all stuck at home anyway, so it seems to be a topical dataset.

Goals

My goal was to take this large, unwieldy dataset and present its findings using a sophisticated visualization that is intuitive to use and provides real insight.

Viewers should be able to detect patterns of similar home prices by looking at the map and draw conjectures on what may have caused this topology. They should be able to see an outlier county within a state and be prompted to dig deeper as to why, for example, Washington County in Greenville, MS is priced so much higher than its neighboring counties.

I'd like the visualizations to engage viewers to explore the map and find something new and interesting everytime.

Overview

Zillow Home Value Index (ZHVI): A smoothed, seasonally adjusted measure of the typical home value and market changes across a given region and housing type. Zillow publishes top-tier ZHVI ($, typical value for homes within the 65th to 95th percentile range for a given region) and bottom-tier ZHVI ($, typical value for homes that fall within the 5th to 35th percentile range for a given region). Zillow also publishes ZHVI for all single-family residences ($, typical value for all single-family homes in a given region), for condo/coops ($), for all homes with 1, 2, 3, 4 and 5+ bedrooms ($), and the ZHVI per square foot ($, typical value of all homes per square foot calculated by taking the estimated home value for each home in a given region and dividing it by the home‚Äôs square footage).

Details

The data for the visualizations has been sourced from the ZHVI All Homes dataset from Zillow.

"The Zillow Home Value Index (ZHVI), built from the ground up by measuring monthly changes in property-level Zestimates, captures both the level and appreciation of home values across a wide variety of geographies and housing types (e.g. all 1-bedroom condominiums in ZIP Code 98101)."

Source: https://www.zillow.com/research/zhvi-methodology-2019-highlights-26221/

Some of the more relevant columns are listed below:

  1. RegionName: County Name
  2. SizeRank: Ranking in Terms of Geographical Size
  3. State: State Name abbr.
  4. StateCodeFIPS: 2 digit State Code
  5. MunicipalCodeFIPS: 3 Digit Municipal Code
  6. YYYY-MM-DD (several columns): Estimated market rate for day DD & month MM of year YYYY

Wrangled

The very wide dataset was first wrangled in Tableau and then in Excel.

Tableau was used to take all the date columns and "pivot" them into the rows such that the dataset is now narrow but much deeper. Excel was used to fill in the missing "County ID" data by merging "State Code FIPS" and "Municipal Code FIPS" with the appropriate trailing 0's, as shown below (in pseudocode).

COUNTY ID = CONCATENATE(TEXT(STATE FIPS, "00"), TEXT(MINICIPAL FIPS, "000"))

Encoding

Each county is encoded positionally on the above choropleth map - the geographic location of the county is encoded by its position on the map. The color within each county encodes latest median home value (incidentally, 2020).

Interactivity

Hovering over a county will cause a tooltip to appear, with details about the county, metro and state name. It also provodes the numerical value of median home value and displays size rank.

Additionally, a contextual line chart also appears, to provide temporal details for the home value, ranging from as far back as 1996 to 2020.

Conclusion

It can be seen that the US has a diverse range of home values across its topography. The West of the US seems to have higher valued homes than the East. We can also notice patterns such as clusters of expensive counties in CA ans a strip of expensive counties lined along the East Coast.

The line charts for several different counties show a steep drop in price around 2008, which is reflective of the US Housing Market Crisis of 2008.

Zillow seems to be missing data for a lot of counties down the vertical middle of the US.
